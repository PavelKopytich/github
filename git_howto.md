# Помощь с GIT

Создание репозитория
```sh
git init
```
Добавить
```sh
git add 
```
Зафиксировать изменения
```sh
git commit -m "message text"
```
log в полном виде
```sh
git log
```
log в кратком виде
```sh
git log --oneline
```
Перемещение по веткам
```sh
git checkout <имя ветки>
```
Отображение всех веток
```sh
git branch
```
