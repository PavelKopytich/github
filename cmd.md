# Подсказки по командной строке

Команда смены директории
```sh
cd c:\folder_name
```

Команда отображения текущей директории
```sh
pwd
```

листинг текущуй директории
Windows:
```sh
dir
```
Linux, MacOs:
```sh
ls
```

Удаление файл в Windows:
```sh
del <filename>
```
в Linux, MacOs:
```sh
rm <filename>
```
